# 🎯 AI Placement Assistant

AI Placement Assistant is a Streamlit-based web application that helps students prepare for placements using AI-powered features. It provides resume analysis, ATS score calculation, interview question generation, skill gap analysis, resume suggestions, and application tracking.

## Features

* 👤 Student Profile Management
* 🏢 Placement Tracker
* 📊 Dashboard with Charts
* 📄 Resume Upload and Analysis
* 🎯 ATS Score Calculation
* 🤖 AI Interview Question Generation
* 🧠 AI Skill Gap Analysis
* 📝 AI Resume Suggestions
* 📋 AI ATS Feedback
* 🛣 Learning Roadmap

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas
* Matplotlib
* PDFPlumber
* Google Gemini API

## Installation

Clone the repository:

```bash
git clone <repository-link>
cd AI-Placement-Assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
